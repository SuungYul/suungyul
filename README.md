### What I've Built

- **통합 분석 데이터 플랫폼** — 개발·운영·공공·마케팅·의사정보 등 5개 소스를 약 8GB 단일 분석 DB로 통합하고 Airflow pull 방식 ETL·증분적재 구성
- **백엔드 성능 엔지니어링** — API 6분 → 0.2초, 핵심 쿼리 92초 → 0.3초, 분석 롤업 46초 → 0.02초
- **세그먼트 분석 엔진** — RFM과 장바구니 연관분석으로 교차판매 타겟을 도출하고 캠페인 조건빌더로 연결
- **이종 도메인 확장** — 단일 도메인 플래그로 매출 중심 제품을 검사건수 중심 도메인까지 확장하고 약 1,300만 행 파이프라인 구성
- **대용량 업로드 파이프라인** — 병원 자동매핑 150초 → 7초, NDJSON 스트리밍 진행률, PII 스캔·차단 적용
- **행정구역 GIS 리매핑** — 공공데이터 간 갱신 시차를 좌표 기반으로 보정해 개편 지역 데이터 불일치 해소
- **자체 크롤링 파이프라인** — 외부 솔루션을 무중단 전환·즉시 롤백 구조로 완전 대체, 86분 → 13분 및 연 1,200만원 절감
- **인프라 현대화** — 사내 최초 GitLab CI/CD 도입, AWS EC2 ARM·RDS Blue/Green 무중단 이관
